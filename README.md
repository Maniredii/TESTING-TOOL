# Automated Website Testing Platform

This is a monorepo containing the Automated Website Testing Platform.

## Structure
- `apps/client`: React 19 Frontend
- `apps/server`: Node.js/Express Backend
- `packages/automation`: Playwright core engine
- `packages/shared`: Shared types and utilities
- `packages/ui`: Reusable UI library

## Setup
```bash
npm install
npm run build
npm run dev
```
